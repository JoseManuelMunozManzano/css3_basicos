```

  border-radius: 50px;
}
```

## ¿Qué se necesita para aplicar Responsive Design?

Ver directorio `16-Responsive_Design`.

Las tres reglas esenciales son:

- Aplicar viewport
- Utilizar porcentajes en contenedores (cajas, header, section...)
- Aplicar Media Query
